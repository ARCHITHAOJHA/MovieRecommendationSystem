# MovieRecommendationSystem

## 🎬 Movie Recommendation System

The Movie Recommendation System is a web-based application developed using Python and Streamlit that suggests movies to users based on their selected preferences. The system uses a content-based filtering approach to recommend movies similar to the one chosen by the user.

The application is built using a preprocessed dataset containing movie information and a similarity matrix generated using machine learning techniques. When a user selects a movie from the dropdown menu, the system calculates similarity scores and displays a list of recommended movies.

To enhance the user experience, the application integrates the OMDb API to fetch real-time movie posters and IMDb ratings. This allows users to visually explore recommendations along with relevant details such as ratings and basic movie information.

The interface is designed to be simple and user-friendly, inspired by modern streaming platforms, where users can easily browse and discover new movies. The application is deployed using Streamlit, making it accessible through a web browser without requiring any additional setup.

### 🔧 Technologies Used

* Python
* Streamlit
* Pandas & NumPy
* Machine Learning (Cosine Similarity)
* OMDb API

### 🎯 Key Features

* Movie recommendations based on similarity
* Display of movie posters
* IMDb ratings integration
* Fast and interactive UI
* Easy deployment and accessibility

This project demonstrates the practical application of recommendation systems and web development, making it suitable for academic projects, portfolios, and real-world applications.
